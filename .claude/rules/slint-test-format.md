# Slint Test Format

## Overview

Tests are headless Rust tests using `slint::testing`. Each test invokes CRUD callbacks on `FlashcardAppLogic` and asserts the resulting model state — no event loop, no visible window.

Tests live in `lib/flashcard/src/lib.rs` under `#[cfg(test)]`.

---

## Prerequisites

### 1. Enable the testing feature in `lib/flashcard/Cargo.toml`

```toml
[dev-dependencies]
slint = { workspace = true, features = ["testing"] }
```

### 2. Add `FlashcardTestWindow` to `lib/flashcard/ui/flashcard_lib.slint`

```slint
// Minimal component used only in tests to host FlashcardAppLogic as a global.
export component FlashcardTestWindow { }
```

This makes `FlashcardAppLogic<'a>: slint::Global<'a, FlashcardTestWindow>` auto-generated by Slint's `include_modules!()`, allowing `init(&window)` to accept it.

## Test Naming

Function names: `[page_name]_[crud_operation]_[subject]`

| Function name | Conceptual name |
|---|---|
| `study_page_create_stack` | `StudyPage.Create` |
| `study_page_create_card` | `StudyPage.Create` |
| `study_page_read_list` | `StudyPage.Read` |
| `study_page_update_card` | `StudyPage.Update` |
| `study_page_reorder_cards` | `StudyPage.Update` |
| `study_page_delete_card` | `StudyPage.Delete` |
| `study_page_delete_stack` | `StudyPage.Delete` |
| `study_page_persistence_round_trip` | `StudyPage.Persistence` |

## Test Goals Header

Start each `#[cfg(test)]` module with a comment block listing covered tasks and what each one adds:

```rust
// # Test Goals
// Page: StudyPage
//
// # [Task 2.1.1 + 2.1.2]:
// - Confirming stack name creates a new FlashcardStackModel appended to flashcard-list
//
// # [Task 2.6.4]:
// - Confirming stack deletion removes the stack at selected-stack-index
// - selected-stack-index resets to -1 after deletion
```

Add new `# [TaskID]:` entries as new tasks are tested — do not create a new test module per task.

## Standard Setup

```rust
#[cfg(test)]
mod tests {
    use slint::Model;
    use super::flashcard::{FlashcardTestWindow, FlashcardAppLogic};

    fn setup() -> FlashcardTestWindow {
        slint::testing::init();
        let window = FlashcardTestWindow::new().unwrap();
        super::init(&window);
        window
    }

    fn seed_stack(logic: &FlashcardAppLogic, name: &str) {
        logic.invoke_stack_create_confirmed(name.into());
    }

    fn seed_card(logic: &FlashcardAppLogic, jap: &str, meaning: &str) {
        logic.invoke_flashcard_add_confirmed(jap.into(), meaning.into());
    }
}
```

## CRUD Operation Templates

### Create

```rust
/// Covers: Task 2.1.1 + 2.1.2 — stack creation appends to flashcard-list
#[test]
fn study_page_create_stack() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();

    logic.invoke_stack_create_confirmed("Hiragana".into());

    let list = logic.get_flashcard_list();
    assert_eq!(list.row_count(), 1);
    assert_eq!(list.row_data(0).unwrap().stackname, "Hiragana");
}

/// Covers: Task 2.2.1 + 2.2.2 — add-card form appends FlashcardModel to selected stack
#[test]
fn study_page_create_card() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();
    seed_stack(&logic, "Stack A");
    logic.set_selected_stack_index(0);

    logic.invoke_flashcard_add_confirmed("犬".into(), "dog".into());

    let cards = logic.get_flashcard_list().row_data(0).unwrap().flashcards;
    assert_eq!(cards.row_count(), 1);
    let card = cards.row_data(0).unwrap();
    assert_eq!(card.jap_obj, "犬");
    assert_eq!(card.explanation, "dog");
    assert!(!card.known);
}
```

### Read

```rust
/// Covers: Task 2.1.1 + 2.1.2 — list reflects all created stacks in order
#[test]
fn study_page_read_list() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();
    seed_stack(&logic, "Stack A");
    seed_stack(&logic, "Stack B");

    let list = logic.get_flashcard_list();
    assert_eq!(list.row_count(), 2);
    assert_eq!(list.row_data(0).unwrap().stackname, "Stack A");
    assert_eq!(list.row_data(1).unwrap().stackname, "Stack B");
}
```

### Update

```rust
/// Covers: Task 2.3.1 + 2.3.2 — edited field values are reflected in the model
#[test]
fn study_page_update_card() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();
    seed_stack(&logic, "Stack A");
    logic.set_selected_stack_index(0);
    seed_card(&logic, "猫", "cat");

    logic.invoke_flashcard_field_changed(0, "犬".into(), "dog".into());

    let card = logic.get_flashcard_list().row_data(0).unwrap()
        .flashcards.row_data(0).unwrap();
    assert_eq!(card.jap_obj, "犬");
    assert_eq!(card.explanation, "dog");
}

/// Covers: Task 2.6.1 + 2.6.2 — reorder moves card from source to target index
#[test]
fn study_page_reorder_cards() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();
    seed_stack(&logic, "Stack A");
    logic.set_selected_stack_index(0);
    seed_card(&logic, "猫", "cat");
    seed_card(&logic, "犬", "dog");
    seed_card(&logic, "鳥", "bird");

    logic.invoke_flashcard_reordered(0, 2);

    let cards = logic.get_flashcard_list().row_data(0).unwrap().flashcards;
    assert_eq!(cards.row_data(0).unwrap().jap_obj, "犬");
    assert_eq!(cards.row_data(1).unwrap().jap_obj, "鳥");
    assert_eq!(cards.row_data(2).unwrap().jap_obj, "猫");
}
```

### Delete

```rust
/// Covers: Task 2.4.1 + 2.4.2 — delete removes the card at the given index
#[test]
fn study_page_delete_card() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();
    seed_stack(&logic, "Stack A");
    logic.set_selected_stack_index(0);
    seed_card(&logic, "猫", "cat");
    seed_card(&logic, "犬", "dog");

    logic.invoke_flashcard_delete_confirmed(0);

    let cards = logic.get_flashcard_list().row_data(0).unwrap().flashcards;
    assert_eq!(cards.row_count(), 1);
    assert_eq!(cards.row_data(0).unwrap().jap_obj, "犬");
}

/// Covers: Task 2.6.3 + 2.6.4 — delete removes stack at selected index; resets index to -1
#[test]
fn study_page_delete_stack() {
    let window = setup();
    let logic = window.global::<FlashcardAppLogic>();
    seed_stack(&logic, "Stack A");
    seed_stack(&logic, "Stack B");
    logic.set_selected_stack_index(0);

    logic.invoke_stack_delete_confirmed();

    assert_eq!(logic.get_flashcard_list().row_count(), 1);
    assert_eq!(logic.get_selected_stack_index(), -1);
    assert_eq!(logic.get_flashcard_list().row_data(0).unwrap().stackname, "Stack B");
}
```

### Persistence

```rust
/// Covers: Task 2.7 — data survives a simulated restart (load_stacks reads what save_stacks wrote)
#[test]
fn study_page_persistence_round_trip() {
    // First session: create and save
    {
        let window = setup();
        let logic = window.global::<FlashcardAppLogic>();
        seed_stack(&logic, "Saved Stack");
        logic.set_selected_stack_index(0);
        seed_card(&logic, "猫", "cat");
        // save_stacks is called automatically after each mutation via init() callbacks
    }

    // Second session: verify load restores state
    {
        let window = setup();
        let logic = window.global::<FlashcardAppLogic>();
        let list = logic.get_flashcard_list();
        assert_eq!(list.row_count(), 1);
        assert_eq!(list.row_data(0).unwrap().stackname, "Saved Stack");
        let cards = list.row_data(0).unwrap().flashcards;
        assert_eq!(cards.row_count(), 1);
        assert_eq!(cards.row_data(0).unwrap().jap_obj, "猫");
    }

    // Clean up test artifact
    let _ = std::fs::remove_file("stacks.json");
}
```

## Running Tests

```powershell
cargo test -p flashcard                        # all flashcard lib tests
cargo test -p flashcard study_page_create      # specific test by name prefix
cargo test -p flashcard -- --nocapture         # show println! output
```
